== Description ==

WooCommerce Latest Product Sales Info is a WooCommerce add-on that allow you to show the time remaining of last sale of any product if it has sold in the last week.

== Installation ==

To install, follow the steps below:

1. Download the zip file that contain the plugin from this page.
2. Upload the zip file through the 'Plugins' menu in WordPress. (You can also unzip the file and upload it through FTP if you prefer).
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enjoy! :-)
